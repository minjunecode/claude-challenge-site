자동 리포팅 설정 가이드

1시간마다 Claude Code + Codex CLI 사용량이 자동 전송됩니다. 5분이면 끝.

작동 방식

Claude Code / Codex 사용
→
로컬 JSONL 자동 기록
→
1시간마다 집계
→
챌린지 사이트
PC의 스케줄러가 ~/.claude/projects/ + ~/.codex/sessions/ 로컬 파일에서 토큰 수(숫자)만 읽어 전송합니다. OAuth 토큰 불필요. Codex를 사용하지 않으면 Claude만 집계됩니다.

설정 가이드

1
파일 2개 다운로드
아래 두 파일을 모두 다운로드한 뒤, .claude 폴더에 넣어주세요.
challenge-report.py setup-scheduler.bat
a
파일 탐색기를 열고, 주소창에 아래를 붙여넣기 → Enter
%USERPROFILE%\.claude
b
.claude 폴더가 열리면, 다운로드한 파일 2개 모두 이 폴더에 넣어주세요.
C:\Users\내이름 > .claude > challenge-report.py + setup-scheduler.bat
2
닉네임 등록 (1회)
챌린지 닉네임/비밀번호를 등록합니다. 딱 한 번만 하면 됩니다.
a
키보드에서 Win 키를 누르고 "cmd" 입력 → 명령 프롬프트 클릭
b
아래를 복사 → 붙여넣기 → Enter
python %USERPROFILE%\.claude\challenge-report.py
c
닉네임과 비밀번호를 물어봅니다. 챌린지 사이트 계정 정보를 입력하세요.

3
자동 실행 등록
Step 1에서 넣어둔 setup-scheduler.bat 파일을 실행합니다.
a
.claude 폴더에 있는 setup-scheduler.bat 파일을 찾아 우클릭 → "관리자 권한으로 실행" 을 클릭하세요.
b
"이 앱이 변경을 허용하시겠습니까?" → "예" 클릭
c
자동으로 Python을 찾고, 스케줄러를 등록합니다. 아래처럼 나오면 성공!

끝!
이제 Claude Code를 사용하면, 매 정각 사용량이 자동으로 챌린지 사이트에 반영됩니다.
PC를 껐다 켜도 자동 실행은 유지됩니다. 더 이상 할 일이 없습니다!
확인 방법: 챌린지 사이트 → 인증 탭에서 "자동 리포팅 활성" 표시가 나오면 성공!
혹시 자동 실행을 해제하고 싶다면:
Win → "cmd" 검색 → 명령 프롬프트 열기 → 아래 입력 후 Enter
schtasks /delete /tn "ClaudeChallenge" /f
1
스크립트 다운로드
아래 버튼을 눌러 파일을 다운로드한 뒤, .claude 폴더에 넣어주세요.
challenge-report.py 다운로드
a
터미널을 열고, 아래 명령어로 다운로드한 파일을 이동하세요.
mv ~/Downloads/challenge-report.py ~/.claude/challenge-report.py
~ > .claude > challenge-report.py  ← 여기에 있으면 OK
2
최초 실행 — 닉네임 등록
터미널에서 아래를 실행하세요. 딱 한 번만 하면 됩니다.
a

python3 ~/.claude/challenge-report.py
b

1시간마다 자동 실행 등록

b

crontab -l
