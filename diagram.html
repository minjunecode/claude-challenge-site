2. 사용자 플로우 (처음부터 끝까지)
1단계: 계정 만들기 (최초 1회)
사이트에 접속 → "계정 만들기" 클릭
// 입력 예시
닉네임: Mj (이니셜 or 닉네임)
비밀번호: Mj1234 (자유)
→ [가입하기] 클릭
2단계: 로그인
가입한 정보로 로그인. localStorage에 저장되어 창 닫아도 유지됨.
닉네임: Mj
비밀번호: Mj1234
→ [로그인] 클릭 → 대시보드 진입
3단계: OAuth 토큰 찾기 (Claude Code 사용자)
내 컴퓨터에서 Claude Code의 OAuth 토큰을 찾습니다.
// Windows 명령 프롬프트(cmd)에서:
C:\Users\내이름> type %USERPROFILE%\.claude\.credentials.json

// Mac/Linux 터미널에서:
$ cat ~/.claude/.credentials.json

// 출력 결과에서 accessToken 값을 복사:
{"claudeAiOauth":{"accessToken":"sk-ant-oat01-xxxxx...xxxxx", ...}}
4단계: 토큰 등록 (최초 1회)
[인증] 탭 → 자동 인증 섹션에 토큰 붙여넣기. 한번 등록하면 서버에 저장되어 영구 유지.
// 토큰 입력란에 붙여넣기:
sk-ant-oat01-2cHWeyOIZvjLA4e_WfOGr7djvd...

→ [등록] 클릭
→ "토큰 등록 완료! 사용량: 5시간 0%, 7일 71%"
5단계: 끝! 자동으로 인증됨
서버가 30분마다 자동으로 사용량을 체크. 95% 이상이면 자동 인증 → 포인트 적립.
// 서버 자동 로직 (사용자가 할 일 없음)
매 30분 → Anthropic API 호출 → 사용량 확인
5시간 세션 ≥ 95% → 세션 인증 자동 기록 (+1pt)
7일 주간 ≥ 95% → 주간 인증 자동 기록 (+5pt)
3. 인증 방식 비교: 자동 vs 수동

AUTO 자동 인증

토큰 1회 등록 후, 서버가 알아서 체크합니다.
  • ✓ Claude Code 사용자만 가능
  • ✓ 30분마다 서버가 자동 폴링
  • ✓ 95% 이상 도달 시 자동 기록
  • ✓ 한번 등록하면 신경 쓸 일 없음
  • ✓ 대시보드에 파란 auto 뱃지

MANUAL 수동 업로드

사용량 스크린샷을 직접 올립니다.
  • ✓ 누구나 가능 (토큰 불필요)
  • ✓ 파일 선택 / Ctrl+V / 드래그앤드롭
  • ✓ 이미지 → Google Drive 저장
  • ✓ 세션: 하루 3회, 2시간 간격
  • ✓ 대시보드에 회색 manual 뱃지

7. 토큰 찾기 가이드 (상세)
1
Claude Code가 설치되어 있는지 확인 (터미널에서 claude 실행 가능해야 함)
C:\Users\내이름> claude
╭────────────────────────╮
│ ✻ Welcome to Claude... │
╰────────────────────────╯
2
토큰 파일 내용 확인 (명령 프롬프트에서 아래 명령어 실행)
C:\Users\내이름> type %USERPROFILE%\.claude\.credentials.json
3
출력에서 accessToken 뒤의 sk-ant-oat01-... 로 시작하는 긴 문자열을 복사
{"claudeAiOauth":{"accessToken":"sk-ant-oat01-xxxxxxx...xxxxxxx","refreshToken":"sk-ant-ort01-..."}}

↑ 이 초록색 부분만 복사 (따옴표 제외)
4
사이트 [인증] 탭 → 토큰 입력란에 붙여넣기 → [등록] 클릭. 끝!
8. 주의사항
⚠️
토큰은 Claude Code 전용. claude.ai 웹/데스크탑 앱의 사용량과는 별도의 한도입니다. 사이트에 표시되는 사용량은 Claude Code 사용량만 반영합니다.
🔄
토큰은 만료됩니다. 만료 시 터미널에서 claude 한번 실행하면 자동 갱신됩니다. 이후 사이트에서 [토큰 변경] 버튼으로 새 토큰을 입력하세요.
🔒
토큰 보안. OAuth 토큰에는 user:inference 스코프가 포함되어 있어 이론적으로 구독 사용이 가능하지만, 과금/비밀번호 접근은 불가합니다.
📱
Apps Script 코드 수정 시 반드시 "새 배포"를 해야 반영됩니다. 저장만으로는 안 됩니다.
🌐
사이트 캐시. GitHub Pages 변경 후 반영 안 되면 Ctrl+Shift+R 또는 URL 뒤에 ?t=1 추가.
