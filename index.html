🖥 여러 PC 사용량 자동 합산

  • 같은 계정으로 2대 이상 PC에 리포터 설치 시 자동 합산
  • 각 PC는 최초 실행 시 고유 machine_id를 자동 생성 (저장: ~/.claude/challenge-machine-id)
  • 서버가 (닉네임, 날짜, machine_id) 기준으로 행 분리 → 대시보드 조회 시 합산된 값으로 표시
  • 한 PC가 보고 실패해도 다른 PC 데이터는 그대로 유지

  • 기존 v2.0 리포터(구 py)는 그대로 동작 — 업그레이드 강제 X
