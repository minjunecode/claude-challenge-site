What is this

구독료만큼 쓰고 있나요?

Claude Max는 월 $100~$200의 프리미엄 AI 구독입니다.
하지만 대부분의 사용자는 한도의 절반도 채우지 못합니다.
이 챌린지는 사용량 한도를 매일 채우는 것을 목표로 합니다.

How it works

참여 방법

가입부터 자동 인증까지, 5분이면 충분합니다.

1

계정 만들기

챌린지 사이트에 접속해서 닉네임과 비밀번호를 만드세요.

닉네임: Mj (이니셜이나 닉네임)
비밀번호: 자유롭게 설정
2

OAuth 토큰 찾기

내 PC에 저장된 Claude Code의 OAuth 토큰을 찾습니다. 명령 프롬프트(cmd)에서 아래 명령어를 실행하세요.

// Windows
type %USERPROFILE%\.claude\.credentials.json

// Mac / Linux
cat ~/.claude/.credentials.json

// 출력에서 accessToken 값을 복사
{"claudeAiOauth":{"accessToken":"sk-ant-oat01-xxxxxxx..."}}
3

토큰 등록 (1회)

사이트 [인증] 탭에서 복사한 토큰을 붙여넣고 [등록]을 누르세요. 이것으로 끝입니다.

// 토큰 입력란에 붙여넣기
sk-ant-oat01-2cHWeyOIZvjLA4e_WfOGr7...

→ [등록] 클릭 → 완료!
4

자동으로 인증됩니다

서버가 30분마다 사용량을 자동 체크합니다.
5시간 세션 95% 이상 → 세션 인증 (+1pt)
7일 한도 95% 이상 → 주간 인증 (+5pt)

토큰 등록 후에는 아무것도 할 필요가 없습니다.

FAQ

자주 묻는 질문

OAuth 토큰이 뭔가요? API Key인가요?
아닙니다. OAuth 토큰은 Claude Max 구독의 사용량을 읽기만 할 수 있는 토큰입니다. API Key와 달리 과금이 발생하지 않고, 비밀번호나 결제 정보에 접근할 수 없습니다. 사용량 조회 전용입니다.
Claude Code가 뭔가요?
Anthropic이 만든 AI 코딩 어시스턴트입니다. 터미널에서 claude 명령어로 실행하며, Claude Max 구독에 포함되어 있습니다. 이 챌린지는 Claude Code의 사용량 한도를 추적합니다.
Claude Code를 안 쓰면 참여할 수 없나요?
자동 인증은 Claude Code 사용자만 가능하지만, 수동 인증도 지원합니다. claude.ai 웹/앱의 사용량 스크린샷을 직접 업로드하면 됩니다. Ctrl+V 붙여넣기, 드래그앤드롭 모두 가능합니다.
토큰을 등록하면 내 계정이 위험한가요?
토큰으로 할 수 있는 것은 사용량 읽기뿐입니다. 계정 비밀번호 변경, 결제 정보 접근, 구독 취소 등은 불가능합니다. 토큰이 만료되면 자동으로 무효화되며, 사이트에서 언제든 변경할 수 있습니다.
토큰을 등록한 후에 해야 할 일이 있나요?
없습니다. 서버가 30분마다 자동으로 사용량을 체크하고, 조건이 충족되면 인증을 기록합니다. 토큰이 만료되었을 때만 갱신해주면 됩니다. (터미널에서 claude 한번 실행하면 자동 갱신)
사이트에 보이는 사용량은 무엇인가요?
Claude Code 전용 사용량입니다. claude.ai 웹/앱의 사용량과는 별도의 한도입니다. Claude Max는 웹과 Code가 각각 독립적인 한도를 가지고 있습니다.
